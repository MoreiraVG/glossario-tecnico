TAGS HTML

1. CRIAR CARDS<H1>

Cards são caixas visuais usadas em sites para organizar informações de forma separada e destacada, como se fossem “blocos” com título, imagem e texto. Eles ajudam a deixar o conteúdo mais limpo e fácil de entender.

2. GRUPO DE CARDS <Div class="grupo-card">

Um grupo de cards é um conjunto de caixas organizadas lado a lado (ou em grade), geralmente usado para exibir informações de forma visual, como produtos, serviços, perfis, etc.

3. CRIAR UM MENU<UL LI>

<ul> é uma lista não ordenada (unordered list).
<li>são os itens da lista.
Você usa esses elementos para criar menus de navegação com links.

4. Colocar imagem <img>

É uma caixa genérica em HTML. Ela não tem estilo nem significado por si só, mas serve para agrupar outros elementos (como texto, imagens, links, menus, etc.).

5. colocar um rodapé <footer>

O footer, serve para fazer rodapé ao final da página, onde geralmente é posto informações de contato ou desenvolvimento

6. colocar uma logo <class>

Usando o class pra classificar as tags para uma formatação mais fácil.

7. colocar um botão <button>

O botão é um elemento HTML usado para interagir com o usuário — ele pode executar ações como enviar formulários, abrir links, rodar scripts, ou ativar funções em JavaScript.

8. como destacar texto <strong>

A tag é um elemento HTML usado para dar ênfase forte a um texto, indicando que aquela parte é importante. Visual → Normalmente deixa o texto em negrito (mas isso pode mudar com CSS). Semântica → Indica para navegadores e leitores de tela que o conteúdo é importante, ajudando na acessibilidade e SEO.

9. formularíos <imput>

A tag é usada no HTML para criar campos interativos em formulários, onde o usuário pode digitar, selecionar ou enviar informações.

10. quebrar de linha <br>

A <br> (break line) é uma tag HTML usada para quebrar a linha no texto, funcionando como se você pressionasse Enter no teclado.
